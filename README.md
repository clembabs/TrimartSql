
# TrimartSq;

Trimart App, a CRUD(Create,Edit,Update,Delete) application was built with Flutter Crossplatform (Android and iOS) Framework with 
- Getx for state management and Navigations [Getx](https://pub.dev/packages/get)
- Sqfentity [Sqfentity](https://pub.dev/packages/sqfentity) to interact with the sqlite database from your Flutter app.
- User friendly UI/UX.


## Features

- CRUD functionalities
- Navigation with Animations
- Local Storage (Products)
- Image Upload

## Set Up

### Environment Variables

Create `api_key.dev.json`  file in the project root directory and add the following keys

`db_name`

### Set up DB
- To set up DB visit the sqfentity [Sqfentity](https://pub.dev/packages/sqfentity) documentation

## Run Locally

### Enter command

```bash
   flutter run --dart-define-from-file=api_keys.dev.json
```
